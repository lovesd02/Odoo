Moneris Checkout Payment

Moneris Solutions Corporation, commonly known as Moneris, is one of Canada's leading payment processing companies. It provides payment processing services, including debit and credit card processing, online and mobile payment solutions, and other related services to businesses of all sizes.

Moneris Checkout gives e-commerce merchants a simple and secure way to process payments by integrating a Moneris-hosted payment module into the merchant checkout page.

This module provides functionality for customers to make payments using Moneris Checkout.

Odoo Provider Configuration

Configure the credentials of Moneris in the Odoo Moneris Payment Provider.

Moneris Checkout & Odoo Provider Configuration

For automatic Authorize & Capture transactions, Make sure you have enabled the "Purchase" option in Moneris Checkout Configuration.

Note: If you have selected the "Purchase" Option from Moneris, disable the "Manually Capture" flow on the Odoo Moneris Provider.



  For Authorize & Pending Capture Transactions, Make sure you have enabled the "Preauthorization" option in Moneris Checkout Configuration.

Note: If you have selected the "Preauthorization" Option from Moneris, enable the "Manually Capture" flow on the Odoo Moneris Provider.

   
   



  For Save Tokenization, Make sure you have enabled the "Tokenization Card" option in Moneris Checkout Configuration.

Note: If you have selected the "Tokenization Card" Option from Moneris, enable the "Allow Saving Payment Methods" option on the Odoo Moneris Provider.

   
   
eCommerce Order Transaction

Shop the item and select Moneris payment provider and click on the "Pay Now" button, system will open the Moneris Checkout screen in Odoo to make a Payment.

Fill credit card details and click on "Checkout" button.

 



The transaction has been successfully processed on Odoo side.

 



  The payment transaction has been approved on Moneris side.  

   



  Check the transaction details on Odoo's side.  

Save Payment Method

Select "Save my payment details" when the make a payment using Moneris Provider and Click on "Pay Now" button.

Payment Token will be generated after successfully done the transaction on the Moneris side.

  
Payment Using Tokenization

Select payment token and click on "Pay Now" button. Shop the item and select payment token and click on the "Pay Now" button.

    

The transaction has been successfully processed on Odoo side.

 

The payment transaction has been approved on Moneris side.  

 

Check the transaction details on Odoo's side.  

 
Authorize Transaction

Select payment token and click on "Pay Now" button. Shop the item and select payment token and click on the "Pay Now" button.

The transaction has been Authorized on Odoo side.

 

The payment transaction has been authorized on Moneris side.  

 

Check the transaction details on Odoo's side.  

 
Capture Transaction

Go to the authorized transaction sale order and click on the "Capture Transaction" button to manually capture the transaction.

  

The payment transaction has been captured on Moneris side.  

 

Check the transaction details on Odoo's side.  

 
Moneris Void Transaction

Go to the Moneris captured transaction sale order and click on the "Void Transaction" button to cancel the transaction.

  

The payment transaction has been canceled on Moneris side.  

 

Check the canceled transaction details on Odoo's side.  

 

Frequently Asked Questions

Does this module allow the user to capture payment manually after authorize?

Yes, it will allow for customer to capture payment manually.

Can we pay an order using a saved token?

Yes, this module is able to make payments using saved tokens.

Can we void the captured transaction using this module?

Yes, this module is able to make void transaction.

Does this module allow the user to pay an order partially?

No, this module does not provide this feature.
